# Workout Calendar

A mobile-friendly workout tracker that can be deployed to GitHub Pages.

## GitHub Pages setup

1. Push this folder to a GitHub repository.
2. In the repository settings, enable GitHub Pages using the `GitHub Actions` source.
3. Push to the `main` branch.
4. Open the Pages URL shown in the Actions tab and add it to your Android home screen from Chrome.

## Files

- `index.html` - the app UI
- `manifest.json` - PWA metadata
- `sw.js` - offline caching
- `icon.svg` - app icon
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow
